Comparison rubric

Five axes — pick the one that scores best for AXA's day-to-day operator workflow. No "right" answer; this is taste under constraint.

Axis Atelier Dossier Copilot
Density · scan-ability Highest — every pixel works hard, zero ornament. Calm — readable but uses macro whitespace. Editorial split — chat steals attention from canvas.
Trust feel Tactical, technical — feels like a control room. Archival, signed-record — feels auditable. Premium, modern — feels like an Apple product.
Authoring ergonomics Graph editor (canvas + nodes). Linear stepper (no canvas). Conversational with diff overlay.
Risk of skill mismatch Looks scary to non-technical reviewers. Could feel under-powered to engineers. Could feel too consumer for an internal tool.
Implementation cost Low · system fonts, zero JS animation. Low · standard CSS, IntersectionObserver. Medium · custom fonts, mesh gradients, GPU motion.
