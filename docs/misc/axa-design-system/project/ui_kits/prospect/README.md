# Apollo — B2C Prospect UI Kit

Recreation of axa.fr's marketing/prospect surface. Based on the Apollo storybook (`apps/apollo-stories`).

**Signature moves**
- Pill (`100vmax`) primary CTAs in AXA blue.
- Publico Headline serif on H1 (Bold) and Publico Light on H2 — editorial feel.
- Hero photography with deep AXA-blue gradient overlay, eyebrow + serif headline + lead.
- 4-up product grid with Material Symbols icons in AXA blue.
- "Pro" CTAs use orange (`#BC4C2D`) to differentiate from consumer flows.

**Components**
- `Header` — sticky white nav with logo + 4 nav items
- `Hero` — full-bleed photo + serif H1 + dual CTA
- `ProductCard` / `ProductGrid` — icon, title, desc, link, optional "Populaire" tag
- `QuoteFlow` — 4-step modal (Vous → Véhicule → Profil → Devis), stepper bar, success state
- `Footer` — dark blue, 4 columns + legal

**Click-thru:** Click "Demander un devis" in the hero to open the 4-step quote modal.
